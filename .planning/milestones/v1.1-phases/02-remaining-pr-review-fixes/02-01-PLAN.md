The return statement is unchanged. nw.get_native_namespace accepts a nw.LazyFrame or nw.DataFrame and returns the underlying native module (e.g., polars). nw.from_dict creates an eager frame in that backend; .lazy() wraps it as a LazyFrame.

Also add inline documentation to the IbisCheckBackend delegation block in __call__. Above the `try: import ibis as _ibis` block, add a comment block explaining:
- Why delegation exists: user-defined ibis check functions expect IbisData(table, key), not NarwhalsData. This is a backward-compat shim.
- Why it is transitional: user checks written for Polars receive a raw pl.LazyFrame (key is dropped). Neither contract is ideal.
- TODO: apply() should unwrap NarwhalsData to the type the check function expects (via type annotation inspection), making IbisCheckBackend delegation unnecessary.

The `import ibis as _ibis` alias (underscore prefix, guarded optional import) is intentional convention — do not change.

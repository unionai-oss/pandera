<objective>
Make all 20 failing drop_invalid_rows tests pass with zero duplicate computation by: (1) changing apply() to return nw.Expr for native=False and element_wise paths, (2) adding postprocess_expr_output() that stores ONLY the nw.Expr and defers failure_cases computation — no wide table built during the check loop, (3) replacing drop_invalid_rows() with a narwhals-idiomatic nw.all_horizontal accumulation that builds ONE wide table from all stored nw.Expr entries, and (4) updating failure_cases_metadata() to reconstruct per-check failure_cases from the stored nw.Expr when SchemaErrors is raised.

Key design: when drop_invalid_rows=True, per-check failure_cases are NEVER computed (zero waste); when SchemaErrors is raised, failure_cases_metadata() computes them exactly once from the stored exprs. The two paths — drop_invalid_rows vs SchemaErrors — are mutually exclusive, so no expr is evaluated twice.

Purpose: Eliminates backend-specific isinstance checks and IbisSchemaBackend delegation from drop_invalid_rows — pure narwhals works identically for polars lazy and ibis.
Output: checks.py and base.py updated (no changes to container.py or components.py needed), 20 previously-failing tests now pass, xfail parity test promoted to passing.
</objective>

            selector = nw.col(key or "*")
            try:
                expr = selector.map_batches(self.check_fn, return_dtype=nw.Boolean)
                return frame.with_columns(expr.alias(CHECK_OUTPUT_KEY))  # CHANGE: return expr
            except NotImplementedError:
                raise NotImplementedError(...)

            native_frame = nw.to_native(frame)
            out = self.check_fn(native_frame, key)
            return self._normalize_native_output(out, check_obj)  # UNCHANGED

            # native=False

       def postprocess_expr_output(
           self,
           check_obj: NarwhalsData,
           expr: nw.Expr,
       ) -> CheckResult:
           """Postprocesses nw.Expr check output into a CheckResult.

           Stores the (possibly ignore_na-modified) expr as check_output and
           defers failure_cases computation entirely — no wide table is built
           during the check loop.

           When drop_invalid_rows=True: failure_cases are never needed, so
           this avoids all per-check materialization.
           When SchemaErrors is raised: failure_cases_metadata() in base.py
           reconstructs them from the stored expr exactly once.

           check_passed is computed via a single-column select+aggregate
           (frame.select(expr) → .select(.all())) — much lighter than
           frame.with_columns(expr) which keeps all original columns.
           """
           if self.check.ignore_na:
               expr = expr | expr.is_null()
           frame = check_obj.frame
           # Single-column aggregate to determine pass/fail — no wide table.
           passed = frame.select(expr.alias(CHECK_OUTPUT_KEY)).select(
               nw.col(CHECK_OUTPUT_KEY).all()
           )
           return CheckResult(
               check_output=expr,   # Store ONLY the expr — failure_cases deferred
               check_passed=passed,
               checked_object=check_obj,
               failure_cases=None,  # Computed later in failure_cases_metadata()
           )
       ```

       def drop_invalid_rows(self, check_obj, error_handler):
           """Remove invalid rows — pure narwhals, no backend delegation.

           Collects nw.Expr check_outputs from schema_errors, attaches them
           as boolean columns via a single with_columns call, filters with
           nw.all_horizontal, then drops the temporary columns.
           Works identically for polars lazy frames and ibis tables.
           native=True check_outputs (wide tables) are skipped — only nw.Expr
           entries are processed.
           """
           errors = getattr(error_handler, "schema_errors", [])
           if not errors:
               return check_obj

           check_exprs = [
               (i, err.check_output)
               for i, err in enumerate(errors)
               if isinstance(err.check_output, nw.Expr)
           ]
           if not check_exprs:
               return check_obj

           frame = nw.from_native(check_obj, eager_or_interchange_only=False)
           bool_cols = [f"__check_output_{i}__" for i, _ in check_exprs]
           wide = frame.with_columns([
               expr.alias(f"__check_output_{i}__")
               for i, expr in check_exprs
           ])
           filtered = wide.filter(nw.all_horizontal(*[nw.col(c) for c in bool_cols]))
           result = filtered.drop(bool_cols)

           # Preserve input type: native in -> native out, narwhals in -> narwhals out
           if isinstance(check_obj, (nw.LazyFrame, nw.DataFrame)):
               return result
           return nw.to_native(result)
       ```

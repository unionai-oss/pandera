  <action>
Create `pandera/backends/narwhals/checks.py` using the narwhals translation of `PolarsCheckBackend`.

**Critical implementation notes (from research — do NOT deviate):**

1. **Signature inspection:** Use `inspect.signature()` on the partial directly (Python correctly resolves remaining params through partial). Check `first_param.annotation is NarwhalsData` (identity check, not equality). If the annotation is `inspect.Parameter.empty`, treat as user-defined (pass native).

2. **element_wise:** Use `nw.col(check_obj.key or "*").map_batches(self.check_fn, return_dtype=nw.Boolean)` wrapped in `try/except NotImplementedError`. Re-raise with exact message: `"element_wise checks are not supported on SQL-lazy backends (Ibis, DuckDB, PySpark) because row-level Python functions cannot be applied to lazy query plans. Use a vectorized check instead."`

3. **apply() return for element_wise:** After with_columns + select, the result is a LazyFrame single-column. Return it directly (not a bool).

4. **Multi-column reduction:** `nw.all_horizontal(*[nw.col(c) for c in col_names]).alias(CHECK_OUTPUT_KEY)` — NOT `pl.fold`.

5. **Horizontal concat:** `nw.concat([check_obj.frame.collect(), check_output.collect()], how="horizontal")` — collect BOTH before concat. The filter uses `~nw.col(CHECK_OUTPUT_KEY)` (tilde, NOT `.not_()`).

6. **passed computation:** `results_df.select(nw.col(CHECK_OUTPUT_KEY).all())` — returns a DataFrame, not a scalar.

7. **bool output:** `nw.from_native(pl.LazyFrame({CHECK_OUTPUT_KEY: [check_output]}), eager_or_interchange_only=False)` — wrap with nw.from_native so the rest of the backend sees nw.LazyFrame.

8. **`__call__` flow:** `preprocess(check_obj, key)` → `NarwhalsData(check_obj, key or "*")` → `apply(narwhals_data)` → `postprocess(narwhals_data, check_output)`.

9. **postprocess dispatch:** `isinstance(check_output, nw.LazyFrame)` for LazyFrame path; `isinstance(check_output, bool)` for bool path. Raise TypeError otherwise.

10. **No groupby/query/aggregate:** These methods raise `NotImplementedError` (same as Polars template).

Import only from `narwhals.stable.v1` as `nw` — never bare `narwhals`.
